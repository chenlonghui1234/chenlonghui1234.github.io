CentOS更换yum源

发表于 2019-04-15

1、备份

mv /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.backup

2、下载新的centos-base.repo到/etc/yum.repos.d/

阿里源地址：http://mirrors.aliyun.com/repo/

wget -O /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-5/6/7.repo

3、清除缓存

yum clean all

4、更新本地yum缓存

yum makecache

5、更新yum软件库

yum update -y

6、查看已安装的服务

yum list installed

Linux常用命令整理

发表于 2019-04-15

一、lsof

1、列出本机正在运行的ipv4/ipv6协议服务

lsof -i 4/6

2、列出本机正在运行的tcp/udp服务

lsof -i tcp/udp

3、列出本机1-1024的开放端口

lsof -i :1-1024

4、列出本机用户的进程服务

lsfo -u user

CentOS添加FTP用户

发表于 2019-04-15

1、创建FTP用户test并指定用户家目录

useradd -d /home/test -m test

2、更改用户权限

限定用户test不能登录shell，只能ftp

usermod -s /sbin/nologin test

用户test恢复正常，允许登录shell

usermod -s /sbin/bahs test

更改用户test的主目录

usermod -d /home/test test

3、修改/etc/vsftpd/vsftpd.conf，限制用户只能访问/home/test，不能访问其他地址

禁止穿越到上级目录

chroot_list_enable=YES
